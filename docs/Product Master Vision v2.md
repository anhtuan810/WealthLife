# WealthLife — MASTER_PROJECT_CONTEXT.md

# Project Overview

We are building an iPhone-first premium wealth strategy simulation game called WealthLife.

This is NOT a traditional life simulator.

This is NOT a spreadsheet finance app.

This is NOT a meme-riches game.

WealthLife is a visually immersive strategic simulation game about:
- escaping financial pressure
- building wealth intelligently
- surviving modern economic uncertainty
- balancing ambition and burnout
- gaining long-term freedom

The emotional core is:

“Can the player escape the rat race before stress, debt, burnout, or bad decisions take over?”

The game should feel:
- modern
- premium
- emotionally intelligent
- immersive
- strategic
- aspirational
- tense
- rewarding

This project prioritizes:
- emotional engagement
- visual immersion
- premium mobile-game feel
- addictive progression
- fast iteration

---

# Core Gameplay Loop

Every month:

1. The player receives a career/business/investing/life-pressure event
2. The player chooses between 2–4 decisions
3. Wealth, stress, and progression systems update
4. New opportunities unlock or disappear
5. The player's future changes

The player constantly balances:
- salary vs burnout
- luxury vs investing
- stability vs opportunity
- career vs freedom
- safety vs leverage
- short-term pleasure vs long-term growth

The game should create:
“One more month” addiction.

---

# Starting Point

The game begins at:
18–22 years old.

The player enters adulthood with:
- limited savings
- student debt or financial pressure
- early career uncertainty
- ambition
- potential

This keeps pacing immediately meaningful.

We skip childhood entirely.
